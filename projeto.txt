## Como Executar

1. **Instale as dependências**:
   ```bash
   pip install flask
   ```

2. **Crie a estrutura de pastas**:
   ```
   mkdir -p templates/aluno templates/professor templates/coordenador static/css static/js
   ```

3. **Salve os arquivos** nas respectivas pastas

4. **Execute o sistema**:
   ```bash
   python app.py
   ```

5. **Acesse no navegador**: `http://localhost:5000`
